# YouTube Downloader (Tkinter)

A simple GUI-based YouTube downloader built using Python and Tkinter.

## Features
- Download YouTube videos (MP4)
- Download audio-only (M4A)
- Progress bar
- Simple GUI

## Technologies Used
- Python
- Tkinter
- yt-dlp

## How to Run
1. Install dependencies:
   pip install yt-dlp
2. Run the app:
   python project.py
